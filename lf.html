Flexbox
Flexbox é um modelo de layout do CSS criado para organizar elementos de forma prática e responsiva. Ele facilita o alinhamento, distribuição de espaço e posicionamento de itens dentro de um container. Use o inspecionar e mude as características do container acima.

Display: flex é a propriedade que ativa o Flexbox.
Flex-direction: Define como os itens são organizados, sendo row = linha horizontal, row-reverse = invertida, column = coluna vertical e column-reverse = invertido.
Justify-content: Controla o alinhamento dos itens no eixo principal, com opções como flex-start, flex-end, center, space-between e space-around.
Align-items: Controla o alinhamento dos itens no eixo transversal, com opções como flex-start, flex-end, center, baseline e stretch.
Flex-wrap: Controla se os itens devem quebrar para a próxima linha quando não houver espaço suficiente, com opções como nowrap, wrap e wrap-reverse.
Align-content: Controla o alinhamento das linhas quando há espaço extra no eixo transversal, com opções como flex-start, flex-end, center, space-between e space-around.
Order: Permite alterar a ordem dos itens dentro do container, atribuindo um valor numérico a cada item, onde itens com valores menores aparecem antes dos itens com valores maiores.
Gap: Controla o espaçamento entre os itens no eixo principal.
Align-self: Permite que um item individual tenha um alinhamento diferente dos outros itens, com opções como auto, flex-start, flex-end, center, baseline e stretch.
