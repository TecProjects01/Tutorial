Manipulação de DOM e Eventos
O DOM (Document Object Model) representa os elementos HTML de uma página como objetos que podem ser acessados e modificados com JavaScript.

Eventos são ações que acontecem na página, como cliques, teclas pressionadas ou movimentação do mouse. Podemos usar eventos para executar funções quando o usuário interage com a página.
