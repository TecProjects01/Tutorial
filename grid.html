CSS grid
O CSS Grid é um modelo de layout do CSS criado para organizar elementos em linhas e colunas de forma prática e responsiva. Ele facilita a criação de estruturas mais complexas para páginas web, permitindo controlar posicionamento e espaçamento dos elementos dentro de um container. Use o inspecionar e altere as propriedades do container acima.
Display: grid é a propriedade que ativa o Grid Layout.
Grid-template-columns: Define a quantidade e o tamanho das colunas do grid, podendo usar valores como 100px, auto, 1fr e repeat().
Grid-template-rows: Define a quantidade e o tamanho das linhas do grid.
Gap: Controla o espaçamento entre linhas e colunas do grid.
Justify-items: Alinha os itens horizontalmente dentro de suas células, com opções como start, end, center e stretch.
Align-items: Alinha os itens verticalmente dentro das células, com opções como start, end, center e stretch.
Justify-content: Controla o alinhamento horizontal do grid inteiro dentro do container.
Align-content: Controla o alinhamento vertical do grid inteiro dentro do container.
Grid-column: Define quantas colunas um item ocupará ou em quais linhas de coluna ele ficará.
Grid-row: Define quantas linhas um item ocupará ou em quais linhas de linha ele ficará.
Grid-area: Permite definir uma área específica para um item dentro do grid.
Place-items: Forma resumida de definir align-items e justify-items ao mesmo tempo.
